## Key Design Principles

1. **Clear Hierarchy**: Important information is prominently displayed
2. **Consistent Navigation**: Same header/footer across all pages
3. **Mobile-First**: All layouts work on mobile devices
4. **Accessible**: Proper heading structure and form labels
5. **Conversion-Focused**: Clear CTAs and contact information
6. **Content-Rich**: Each page provides valuable information
7. **SEO-Friendly**: Proper heading structure and content organization
